🖥️ 终端使用完整指南

PowerShell 设置

# 一次性设置（每次打开都需要）
$env:Path += ";C:\Users\xjtul\AppData\Local\Programs\Python\Python312"

# 永久设置（只需执行一次）
[Environment]::SetEnvironmentVariable("Path",
$env:Path + ";C:\Users\xjtul\AppData\Local\Programs\Python\Python312", "User")

基本操作流程

cd C:\Users\xjtul\Biomni
python interactive_mode.py
# 输入问题
# 查看结果
# 输入 quit 退出
❌ 常见问题：
  • 找不到 python 命令：使用上面的环境设置
  • 程序启动慢：首次启动需要10-30秒
  • 编码错误：运行 chcp 65001

🛠️ 故障排除指南

问题1：找不到 python 命令

# 快速修复
$env:Path += ";C:\Users\xjtul\AppData\Local\Programs\Python\Python312"
python --version

问题2：程序启动失败

# 重新安装
pip install biomni --upgrade --force-reinstall

问题3：API连接失败

# 测试连接
python test_zhipu.py

# 检查配置
notepad .env

问题4：编码错误

# 设置编码
chcp 65001
