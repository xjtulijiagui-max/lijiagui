## 🎯 培训课程安排

### 📅 第1天：基础入门（2小时）

**上午（10:00-12:00）：环境准备**
1. **安装验证**（30分钟）
   - 检查Python安装
   - 安装Biomni
   - 配置API密钥
   - 运行测试脚本

2. **基础操作**（1.5小时）
   - 启动交互式模式
   - 学习基本提问
   - 理解输出格式
   - 练习简单任务

**下午（14:00-16:00）：基础应用**
1. **功能体验**（1小时）
   - 尝试Web界面
   - 运行分析脚本
   - 对比三种模式

2. **问题解决**（1小时）
   - 学习故障排除
   - 练习常见问题解决
   - 建立问题日志

### 📅 第2天：进阶应用（4小时）

**上午（9:00-12:00）：专业功能**
1. **药物靶点分析**（2小时）
   - 肺癌药物筛选案例
   - 靶点特征分析
   - ADMET预测
   - 脱靶风险评估

2. **实验设计**（2小时）
   - CCK8实验设计
   - 分组和浓度设置
   - 统计分析方法

**下午（14:00-17:00）：实战练习**
1. **案例演练**（2小时）
   - 完整药物筛选流程
   - 实验设计方案
   - 结果分析报告

2. **问题讨论**（1小时）
   - 学员问题解答
   - 经验分享
   - 最佳实践讨论

## 💡 培训技巧

### 给讲师的建议

**开训前准备：**
1. 确保所有电脑环境正常
2. 提前测试所有脚本
3. 准备备用方案
4. 打印快速指南

**培训过程中：**
1. 先演示，后让学员操作
2. 每30分钟休息一次
3. 鼓励提问和讨论
4. 及时记录学员问题

**培训结束后：**
1. 收集学员反馈
2. 更新培训材料
3. 建立学员群组
4. 提供后续支持

### 给学员的建议

**培训前：**
1. 阅读快速入门指南
2. 检查电脑环境
3. 准备笔记本记录
4. 带好自己的研究问题

**培训中：**
1. 积极参与练习
2. 不怕犯错和提问
3. 记录关键操作
4. 与同学讨论交流

**培训后：**
1. 继续练习使用
2. 应用到实际工作
3. 分享使用经验
4. 提出改进建议
