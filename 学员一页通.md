## 🎯 三种使用方式

### 1️⃣ 交互式模式（推荐）
```
启动：双击 `启动交互模式.bat`
使用：输入问题，查看结果，继续对话
退出：输入 quit
```

### 2️⃣ Web界面
```
启动：双击 `启动Web界面.bat`
访问：http://localhost:5000
使用：浏览器输入问题
```

### 3️⃣ 专门脚本
```powershell

python cck8_experiment_design.py

# 肺癌药物筛选

```

## ⚠️ 遇到问题？

### ❌ 找不到python命令
```powershell
# 临时解决
$env:Path += ";C:\Users\xjtul\AppData\Local\Programs\Python\Python312"
```

### ❌ 程序无响应
- 等待10-30秒（首次启动）
- 检查网络连接
- 重新启动程序

### ❌ API连接失败
- 检查 `.env` 文件中的API密钥
- 确保网络连接正常
- 运行 `python test_zhipu.py` 测试
