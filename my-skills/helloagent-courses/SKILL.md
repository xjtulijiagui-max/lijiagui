## 学习路径推荐

### 按角色推荐

**AI零基础小白**：
I1（剧本杀）→ I7（提示词）→ I3（工具实战）

**产品经理/业务人员**：
I7（提示词）→ I4（产品经理）→ O2（场景创新）

**程序员/技术人员**：
I7（提示词）→ I5（研发提效）→ I6（技术体系）→ I9（全栈落地）

**管理者/高管**：
I7（提示词）→ O4（AI领导力）→ O0/O1（战略规划）

**制造业从业者**：
I7（提示词）→ I8（制造赋能）

### 按层级递进

```
L1 基础认知与启蒙 ──→ I1剧本沙盘、I3前沿实践、I7提示词
         ↓
L2 能力构建与岗位应用 ──→ I2智能体、I4产品经理、I5研发提效
         ↓
L3 技术落地与行业深化 ──→ I6技术体系、I8制造赋能、I9全栈落地
         ↓
L4 组织级战略与变革 ──→ O0/O1战略、O2场景创新、O3/O4变革领导
```

## 常见问答

**Q: 完全没有AI基础，应该选哪门课？**
A: 建议从I1（AI剧本杀）或I7（提示词）开始，都是L1启蒙层课程，不需要技术背景。

**Q: 想快速掌握AI工具搭建，选哪门？**
A: 推荐I2（智能体从入门到精通），学完可以独立搭建智能体应用。

**Q: 企业想进行AI转型，高管应该学什么？**
A: 高管建议学习O系列组织提效课程，特别是O0（AI战略落地）和O4（AI领导力）。

**Q: 制造业适合哪些课程？**
A: 制造业专属课程是I8（AI赋能制造和车间管理），也可以先学I7提示词打基础。

**Q: 课程可以定制吗？**
A: 可以。可按客户行业、团队能力层级、关键业务痛点进行组合式定制交付，也可提供咨询式陪跑、教练式辅导和智能体定制开发服务。
